# Research Papers

Papers that informed Motoko's design and point to future research directions.

## Agent Harnesses & Architectures

| Paper | arXiv ID | Relevance to Motoko |
|---|---|---|

| Meta-Harness: End-to-End Optimization of Model Harnesses | [2603.28052](https://arxiv.org/abs/2603.28052) | Harness optimization methodology; directly applicable to Motoko's edit modes and tool dispatch |
| Agentic Code Reasoning | [2603.01896](https://arxiv.org/abs/2603.01896) | Reasoning patterns for code-modifying agents; informed parse and tool-runtime design |
| AutoAgent: A Fully-Automated and Zero-Code Framework for LLM Agents | [2502.05957](https://arxiv.org/abs/2502.05957) | Agent framework design patterns and automation strategies |
| DR-Venus: Towards Frontier Edge-Scale Deep Research Agents with Only 10K Open Data | [2604.19859](https://arxiv.org/abs/2604.19859) | Research agent architecture; extension system design |
| Barbarians at the Gate: How AI is Upending Systems Research | [2510.06189](https://arxiv.org/abs/2510.06189) | Research methodology in AI era; experimental design philosophy |
